# 📌 01 - What is MongoDB

## 1️⃣ What is MongoDB?

MongoDB is a **NoSQL, document-oriented database** that stores data in **JSON-like documents (BSON format)** instead of tables and rows.

- Developed by MongoDB Inc.
- Open-source database
- Designed for **high availability, scalability, and performance**
- Stores data in flexible schema format

Unlike traditional relational databases, MongoDB does **not require a fixed schema**.

## 3️⃣ Core Characteristics

### ✅ Document-Oriented
Data is stored as documents (JSON-like format).

### ✅ Schema-Less
Documents in the same collection can have different fields.

### ✅ Scalable
Supports:
- Horizontal scaling (Sharding)
- Replication (Replica sets)

### ✅ High Performance
- Uses indexes
- Supports in-memory storage engine (WiredTiger)

## 4️⃣ Students Collection Example

Database: `trainingDB`  
Collection: `students`

Example document:

```json
{
  "name": "Bhanu",
  "age": 23,
  "cgpa": 8.5
}
````

Another document (schema flexibility):

```json
{
  "name": "Anil",
  "age": 24,
  "cgpa": 8.9,
  "internship": true
}
```

Notice:

* Second document has an extra field.
* MongoDB allows this (schema flexibility).

## 5️⃣ How MongoDB Stores Data Internally

MongoDB stores data in:

* **BSON (Binary JSON)** format
* BSON supports additional data types like:

  * Date
  * ObjectId
  * Binary
  * Arrays

Example internal structure:

```json
{
  "_id": ObjectId("65fa1234abcd123456789000"),
  "name": "Bhanu",
  "age": 23,
  "cgpa": 8.5
}
```

`_id` is automatically created as a **primary key**.

## 7️⃣ Common Errors & Fixes

### ❌ Error: Cannot connect to MongoDB

Reason:

* MongoDB service not running
* Port 27017 blocked

Fix:

```bash
netstat -ano | findstr 27017
```

Start MongoDB service:

```bash
mongod
```

---

### ❌ Error: Duplicate key error

Reason:

* Attempt to insert document with same `_id`

Fix:

* Remove duplicate `_id`
* Or let MongoDB auto-generate
